# CLV (Closing Line Value) signal 分析

対象: 全 mc 系戦略 4月以降の bets n=6639 (CLV あり)
CLV = (odds_at_bet - closing_odds) / closing_odds
- CLV > 0: 締切時オッズ低下 = sharp money 流入 = picks が市場に評価された
- CLV < 0: 締切時オッズ上昇 = 市場が picks を見送った

## 留意事項

- CLV は **事後 metric** (締切時オッズ必要)、リアルタイム filter には使えない
  → ただし 「過去 N 日 CLV ROI 高い picks 条件」を学習して real-time 適用は可
- 戦略別差は picks 数に偏りあり、絶対 ROI ではなく相対比較
- 採用候補は shadow 並走で検証 (P7 教訓)
